# 🎹 Digital Piano

An interactive virtual piano built using HTML, CSS, JavaScript, and the Web Audio API. Users can play notes using both mouse clicks and keyboard keys, adjust volume, and switch waveforms for different sound effects.

## 🚀 Live Demo

🔗 https://gsanjana10.github.io/Digital-Piano-Live/


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Audio API

 ## 📌 Features

- Interactive piano keys
- Keyboard support
- Multiple waveforms (Sine, Square, Triangle, Sawtooth)
- Volume control
- Octave switching
- Responsive UI
- Real-time audio generation using Web Audio API
